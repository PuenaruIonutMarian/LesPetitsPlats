filterRecipes

Filters recipes based on tags, input value, or both. It checks for matches in the recipes array based on the normalized tags and input value. The filtered recipes are then updated using the updateCurrentRecipes method, and the UI is updated with the filtered recipes using the updateWithFilteredRecipes method.
